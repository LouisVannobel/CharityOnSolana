## Overview du projet
Ce projet est une plateforme de charité basée sur la blockchain Solana permettant aux utilisateurs d'acheter des NFTs pour soutenir des causes sociales. En retour, les donateurs reçoivent des tokens SPL en guise de récompense.

## Technologies utilisées
## Blockchain et Backend
Solana Devnet
Metaplex Candy Machine v2
Anchor Framework
Solana Web3.js
Token Program & Associated Token Account 

## Frontend
React + Vite
Tailwind CSS
Solana Wallet Adapter


Objectifs:
● Développer une plateforme de charité basée sur la blockchain pour soutenir des
causes (par exemple, les animaux abandonnés) en utilisant des NFTs et des tokens
SPL
● Concevoir une collection de NFTs dont les prix varieront entre 1 et 5 SOL.
● Récompenser les utilisateurs avec des tokens SPL pour leurs contributions (10
tokens pour chaque SOL donné).
● Créer une interface utilisateur simple et conviviale pour interagir avec la
plateforme.
Fonctionnalité
● Token SPL de Charité :
○ Créer un token SPL personnalisé pour servir de récompense sur la
plateforme.
○ Les tokens seront distribués automatiquement aux donateurs en fonction
de leurs contributions.
● Collection NFT :
○ Concevoir et déployer une collection de NFTs (par exemple, axée sur les
animaux abandonnés, la protection de l’environnement, ou tout autre sujet
pertinent).
○ Utiliser Metaplex Token Metadata pour associer des métadonnées (image,
description, nom) à chaque NFT.
○ Les NFTs seront proposés à un prix variant entre 1 et 5 SOL.
● Achat de NFT :
○ Les utilisateurs pourront acheter des NFTs pour soutenir la cause.
○ Le paiement (en SOL) sera directement envoyé au portefeuille de charité.
○ Le donateur recevra :
■ Le NFT acheté dans son portefeuille.
■ Des tokens de récompense proportionnels au SOL dépensé (10
tokens par SOL).
● Interface client :
● Achage des NFTs : Acher tous les NFTs de la collection avec leur prix,
image et détails.
● Achat de NFTs : Orir une fonctionnalité permettant aux utilisateurs
d'acheter des NFTs avec du SOL.
● Visualisation du portefeuille : Permettre aux utilisateurs de voir:
○ Les NFTs qu'ils possèdent.
○ Leur solde de tokens de récompense.
